# Gemini Agent Rules

## Core Workflow
이 프로젝트에서 작업할 때는 다음 수칙을 반드시 준수하십시오.

1. **상태 문서 확인**: 작업을 시작하기 전에 항상 `.gemini/project_status.md`를 읽으십시오.
2. **상태 업데이트**:
   - 작업을 시작할 때: `Current Task` 섹션을 업데이트하십시오.
   - 작업이 끝날 때: 해당 항목을 체크(`[x]`)하고 다음 작업을 설정하십시오.
3. **작업 단위 커밋**: 논리적 작업 단위가 끝날 때마다 커밋하십시오.

## Style Guide
- **언어**: 과정 설명 및 커밋 메시지는 **한글**을 사용하십시오.
- **UI/UX**: 모바일 퍼스트, Shadcn UI 및 Tailwind CSS 사용.
