# 프로젝트 상태 및 명세서

## ⚠️ Workflow Rules (작업 수칙)
**이 문서는 프로젝트의 "Single Source of Truth"입니다.**
1. **작업 리스트 설정**: 요구사항을 분석하여 [Backlog]에 작업을 등록합니다.
2. **상태 동기화**: 작업 시작 시 [Current Task]를 업데이트하고, 완료 시 [Completed]로 이동합니다.
3. **과정 기록**: 중요한 의사결정이나 변경 사항은 이 문서에 반영합니다.
4. **커밋**: 작업 단위마다 커밋을 수행합니다.
